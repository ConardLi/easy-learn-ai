# Manus AI Assistant 能力

## 概述
我是一个 AI 助手，旨在使用各种工具和能力帮助用户完成广泛的任务。本文档在尊重专有信息边界的前提下，更详细地概述了我能做什么。

- 从多个来源进行事实核查与信息验证
- 将复杂信息总结为易于理解的格式
- 处理与分析结构化和非结构化数据

### 内容创作
- 撰写文章、报告和文档
- 起草邮件、消息及其他沟通内容
- 使用各种编程语言创建和编辑代码
- 生成故事或描述等创意内容
- 按照特定要求格式化文档

### 问题解决
- 将复杂问题拆解为可管理的步骤
- 为技术难题提供逐步解决方案
- 排查代码或流程中的错误
- 当初始尝试失败时建议替代方案
- 在任务执行过程中适应不断变化的需求

## 工具与接口

### 部署能力

# 高效 Prompting 指南

## Prompting 入门

本文档为与 AI 助手协作时如何创建有效的 prompt 提供指导。一个精心设计的 prompt 可以显著提升你所获得回复的质量和相关性。

## 有效 Prompt 的关键要素

### 具体且清晰
- 明确陈述你的请求
- 包含相关的上下文与背景信息
- 指定你希望回复采用的格式
- 提及任何约束或要求

### 提供上下文
- 解释你为什么需要这些信息
- 分享相关的背景知识
- 在适用时提及之前的尝试
- 描述你对该主题的熟悉程度

### 组织你的请求
- 将复杂请求拆分为更小的部分
- 对多部分问题使用编号列表
- 当询问多件事时对信息进行优先级排序
- 考虑使用标题或分节来组织内容

### 指定输出格式
- 指明偏好的回复长度（简短还是详细）
- 请求特定的格式（要点、段落、表格）
- 提及你是否需要代码示例、引用或其他特殊元素
- 在相关时指定语气和风格（正式、对话式、技术性）

## 示例 Prompt

### 差的 Prompt：
"Tell me about machine learning."（告诉我关于机器学习的事。）

### 改进后的 Prompt：
"I'm a computer science student working on my first machine learning project. Could you explain supervised learning algorithms in 2-3 paragraphs, focusing on practical applications in image recognition? Please include 2-3 specific algorithm examples with their strengths and weaknesses."（我是一名计算机科学专业的学生，正在做我的第一个机器学习项目。你能用 2-3 段解释监督学习算法吗，重点关注图像识别中的实际应用？请包含 2-3 个具体的算法示例及其优缺点。）

### 差的 Prompt：
"Write code for a website."（为一个网站写代码。）

### 改进后的 Prompt：
"I need to create a simple contact form for a personal portfolio website. Could you write HTML, CSS, and JavaScript code for a responsive form that collects name, email, and message fields? The form should validate inputs before submission and match a minimalist design aesthetic with a blue and white color scheme."（我需要为一个个人作品集网站创建一个简单的联系表单。你能编写 HTML、CSS 和 JavaScript 代码，做一个收集姓名、邮箱和留言字段的响应式表单吗？该表单应在提交前验证输入，并符合采用蓝白配色方案的极简设计美学。）

## 迭代式 Prompting

请记住，与 AI 助手协作通常是一个迭代的过程：

1. 从一个初始 prompt 开始
2. 审阅回复
3. 根据哪些有帮助或缺失之处来优化你的 prompt
4. 继续对话以进一步探索该主题

## 为代码进行 Prompting 时

在请求代码示例时，考虑包含以下内容：

- 编程语言及版本
- 你正在使用的库或框架
- 排查问题时的错误消息
- 输入/输出示例
- 性能方面的考量
- 兼容性要求

## 结语

高效的 prompting 是一项随实践而提升的技能。通过做到清晰、具体并提供上下文，你可以从 AI 助手处获得更有价值、更相关的回复。请记住，如果初始回复未能完全满足你的需求，你随时可以优化你的 prompt。
