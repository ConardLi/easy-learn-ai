## ✨ 项目简介

Easy AI 是一个专注于 AI 教育的现代化 Web 应用平台，旨在通过简洁直观的方式帮助用户理解复杂的人工智能概念。平台集成了多个学习模块，提供从基础概念到实践应用的全方位 AI 知识体系。
