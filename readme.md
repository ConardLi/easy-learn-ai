## 项目定位

Easy AI 把复杂的 AI 概念、模型信息、评测基准、教程、提示词样本和行业动态，整理成适合阅读、查询、比较和持续追踪的形式。

它不是一个堆链接的资料库，而是一个经过编辑整理的 AI 学习入口：你可以从一个概念开始理解原理，也可以直接查询模型参数、拆解优秀提示词、追踪最新日报、阅读教程，或者找到可用的 AI 工具。
